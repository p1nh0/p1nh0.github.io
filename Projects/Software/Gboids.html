Gboids - Granular Boids

This MaxMSP patch uses an implementation of Craig Reynoldsâ€™ "boids" algorithm (available in Max Javascript Examples) to drive a granular synthesizer, where each agent represents a grain, scrubbing an audio sample as it moves in a two-dimensional space. This brings a semantic change to the sound synthesis model, like position, grain size, etc. to semantics related to motion and collective behaviour, like inertia, alignment and separation.

December 2010.
